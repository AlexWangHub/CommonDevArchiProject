# CommonDevArchiProject

一个通用的iOS开发底层模板（持续更新）

一、Pod安装的工具framework：

- YYKit
- Colours
- LazyScroll：自己的项目如果用 tableView 写相比直接画 View 开发速度会慢一些，但 tableView 好在可以自动回收。LazyScroll可以实现更方便自定义View的同时，也支持复用机制

二、文件分类分类：

- System：系统文件
- Helper：工具类
- ViewController：控制类
- ViewModel：模型类
- View：视图
- Manage：管理类
- CGI：网络类
- Category：分类

三、基于此类的开发流程

直接 Fork 或者 Download 均可使用。
